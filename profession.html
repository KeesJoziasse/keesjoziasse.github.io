YourSurprise is een bedrijf dat zich richt op het personaliseren van cadeaus. Dit kunnen klanten zelf doen met behulp van ‘De Editor’, een door YourSurprise gecreëerd programma waar klanten zelf tekst en/of plaatjes aan het product toe kunnen voegen.
De openstaande vacatures binnen de IT-sector zijn:
- PHP Developer
- JavaScript Developer

Syntess Software is een producent van ERP- en mobiele software. Het mooie aan ERP-software is dat het softwarepakket ondersteuning biedt voor alle processen binnen het bedrijf in plaats van losse softwareproducten.
De openstaande vacatures binnen de IT-sector zijn o.a.:
- Front-End Developer
- Software Developer
- ERP consultant

Macaw richt zich voornamelijk op data, design en bedrijven helpen met technische oplossingen. Macaw werkt o.a. met het door Microsoft ontworpen platform Power BI. Power BI wordt gebruikt om inzicht te krijgen in data van organisaties en bedrijven.
De openstaande vacatures binnen de IT-sector zijn o.a.:
- Data Scientist
- Cloud Engineer

WebNL is een bedrijf dat zich net als Macaw voornamelijk richt op bedrijven helpen met problemen op het vlak van organisatie, data overzichtelijk maken en bijvoorbeeld sites maken.
De openstaande vacatures binnen de IT-sector zijn:
- Front-End Developer
- Back-End Developer

Oracle is de grootste software ontwikkelaar ter wereld. De producten zijn voornamelijk gericht op informatie managen, gebruiken, delen en beschermen. Oracle is een product- en serviceorganisatie.
De openstaande vacatures binnen de IT-sector zijn:
- Technisch beheerder
- Developers
- Systeemontwikkelaars
- Engineers
